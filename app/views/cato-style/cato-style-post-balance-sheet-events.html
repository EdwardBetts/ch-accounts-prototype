Post balance sheet events

What's this?Turnover policy

Post balance sheet events are favourable (or unfavourable) events which occurred between the balance sheet date and the date the financial statements were authorised for issue.

Adjusting events are where an adjustment to the figures is necessary. For example, if a trade debtor is declared bankrupt after the balance sheet date and the money they owe cannot be recovered.

Non-adjusting events are where no adjustment to the figures is necessary. These events only need to be disclosed as a note, giving the nature of the event and estimated financial impact (or state that an estimate can't be made).
